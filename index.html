Angularjs Notes Series (1)

setting up the environment

  • install the Nodejs
  • install the webserver connect.
    `npm install connect`

    NPM is the node package installer, and it will pull down the files required for the Connect module. >

  • Next, create a new file called server.js*

    var connect = require(‘connect’),
    serveStatic = require(‘serve-static’);
    var app = connect();
    app.use(serveStatic(“../angularjs”));
    app.listen(5000);

  • Installing the Test System

    npm install -g karma

  • Create a folder called angularjs at the same level on the disk as the Node.js installation folder
  • Download the Angularjs libbary and put in the folder just created
  • Getting Bootstrap
    Go to http://getbootstrap.com and click the Download Bootstrap button. You will receive an archive that
    contains JavaScript and CSS files. Copy the following files into the angularjs folder alongside the angularjs file:
    bootstrap-3.0.3/dist/css/bootstrap.css
    bootstrap-3.0.3/dist/css/bootstrap-theme.css
  • Performing a Simple Test
    create a file called test.html
  • Starting the Web Server
    node server.js

  • Load the Test File in your browser

    http://localhost:5000/test.html
